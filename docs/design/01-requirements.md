# 브랜드 & 상품
## 브랜드 정보 조회
### [유저 스토리]
- 사용자는 브랜드의 정보를 조회할 수 있다

### [기능 흐름]
- 로그인 / 비로그인 사용자 가능
- 해당하는 브랜드가 존재하면 정보 반환, 없으면 에러

## 상품 목록 조회
### [유저 스토리]
- 사용자는 상품의 목록을 조회할 수 있다

- 최신순, 가격순, 좋아요순으로 정렬해서 조회할 수 있다
- 상품 목록을 페이징해서 조회할 수 있다

### [기능 흐름]
- 로그인 / 비로그인 사용자 가능
- 특정 브랜드 필터링, 최신순/가격순/좋아요순 정렬, 페이징 옵션을 조합해서 조회
- 조건에 맞는 상품이 없는 경우 빈 리스트 반환
